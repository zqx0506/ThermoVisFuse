This open-source code is directly associated with the manuscript submitted to The Visual Computer, and the implementation corresponds to the core method, network architecture, experimental setup, and baseline pipelines proposed in the paper. To comply with the double-blind review policy and avoid potential disclosure, only the overall framework and non-critical parts of the code are currently released; the core innovative algorithms, training details, complete experimental scripts, and pre-trained weights are temporarily withheld.
After the manuscript is officially accepted for publication, the full version of the code, including complete training/testing scripts, detailed documentation, dataset instructions, and pre-trained models, will be fully released to ensure reproducibility.
If you use this framework or draw inspiration from it in your research, we sincerely appreciate it if you could cite the corresponding manuscript submitted to The Visual Computer. The formal BibTeX citation will be updated in this repository upon acceptance.
